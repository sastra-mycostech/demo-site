Word Add-in Demo

Secure document scanning and redaction workflow for Microsoft Word.

Quick Start

  1. Download manifest.xml and sideload it in Microsoft Word.
  2. Open demo.docx in Word Desktop or Word on the web.
  3. Launch the add-in, run a scan, and apply suggested redactions.

Install in Word Desktop (Windows/Mac)

  1. Open Microsoft Word.
  2. Go to Home or Insert, then click Add-ins or Get Add-ins.
  3. Open the My Add-ins tab.
  4. Select Upload My Add-in (in some versions: My Add-ins > Manage My Add-ins > Upload).
  5. Choose the manifest.xml file downloaded from this demo page.
  6. Click Upload.
  7. If successful, you will see the RuleSentry tab/button on the Ribbon.
  8. Click a button such as Scan Document to open the Task Pane.

Install in Word on the web

  1. Open Word Online and open a document.
  2. Go to Insert > Add-ins > My Add-ins.
  3. Select Upload My Add-in.
  4. Choose manifest.xml.
  5. Click Add, then launch it from My Add-ins or the Ribbon.
